## ⚡ 5-Minute Setup

### Step 1: Get API Key
👉 **Go to:** https://openweathermap.org/api
- Sign up (free account)
- Copy your API key
- ⏱️ **Wait 10-60 minutes for activation**

### Step 2: Install Dependencies
```bash
pip install -r requirements.txt
```

**Or use the automated setup:**
```bash
bash setup.sh
```

### Step 3: Run (Choose One)

#### Option A: Web Dashboard (Recommended) 🌐
```bash
python app.py
```
Then open: **http://localhost:5000**

#### Option B: Command Line 💻
```bash
python weather_analyzer.py
# Paste your API key when prompted
```

## ⚠️ Common Issues & Fixes

### "401 Unauthorized"
**Problem:** API key not activated yet
**Solution:** Wait 10-60 minutes, then retry

### "ModuleNotFoundError: No module named 'requests'"
**Problem:** Dependencies not installed
**Solution:** `pip install -r requirements.txt`

### "Port 5000 already in use"
**Problem:** Another app using Flask's default port
**Solution:** Flask will auto-find another port, or kill the process

### "No data displayed in dashboard"
**Problem:** API key field is empty
**Solution:** Enter your API key and click "Analyze"
